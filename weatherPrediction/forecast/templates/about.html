About This Project

This weather app is designed to provide users with up-to-date weather forecasts using modern web technologies.

Technologies Used

  • 🌐 HTML5 & CSS3 — For structure and styling
  • 🎨 Bootstrap 5 — For responsive design
  • 📊 Chart.js — To visualize forecast data
  • 🧠 Django — Backend logic & templating
  • ☁️ OpenWeatherMap API — Weather data source
